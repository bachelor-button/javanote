

# 第四章 元空间

## 1、不同版本名称

- <1.8 方法区（Method Area）
- ≥1.8元空间（Meta Space）



## 2、元

本身含义：万物初始，一件事情的源头或基本组成部分。

举例：元素、元始天尊、每年1月称为元月、1月1日称为元旦、元认知、元无知、元知识

对比类和对象，类相当于是对象的元信息。



## 3、元空间存储数据说明

- 类信息：类中定义的构造器、接口定义
- 静态变量（类变量）
- 常量
- 运行时常量池
- 类中方法的代码
